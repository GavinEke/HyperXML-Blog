Get-Command | Get-Random | Get-Help
If you don't know what PowerShell is, it is a command-line shell and scripting language from Microsoft based on the .NET Framework. PowerShell is not something new, in fact in November it will be 10 years since the stable release of PowerShell 1.0, and one of the first products to feature PowerShell was Microsoft Exchange 2007. PowerShell is now up to version 5 and the follow table shows the major features included in each version.
Version Feature Highlight
1.0 Initial Release
2.0 PSRemoting
3.0 Workflows
4.0 DSC
5.0 Classes
PowerShell 5 is pre-installed on Windows 10 but if you are running PowerShell 4 on Windows 7 SP1, Windows 8.1, Windows 2008 R2 or Windows Server 2012 R2 you can upgrade to PowerShell 5.0 by downloading Windows Management Framework (WMF) 5.0 from https://www.microsoft.com/en-us/download/details.aspx?id=50395 which includes PowerShell 5.0 but as always make sure you read the System Requirements and Install Instructions before upgrading.
