A few things have changed since starting this blog the latest being upgrading WordPress to version 4.2 but the biggest change change was on the server that is hosting the website, after 2 months of monitoring the sites performance and reliability I have found I did not need as high spec as initially ordered but the best thing about "the cloud" is that I can upscale and downscale as required.
Old Server New Server
CPU Cores 4 1
Memory 2GB 512MB
SSD Drive 40GB 20GB
Data Transfer 600GB 200GB
As you can see from the above table I have downsized the server quite a bit but the advantage of this is the extra computing power was not required and I have gone from paying $25/month to $10/month which will allow me to spend the money on other projects that will become apart of the content of this blog. I have also replaced the built in WordPress commenting system with Disqus as it is fast and highly used allowing people to only requiring one login for commenting on many sites.
